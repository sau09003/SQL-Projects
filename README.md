# SQL Projects Portfolio

## Overview
This repository contains SQL projects that demonstrate database design, data cleaning, data analysis, and query optimization skills. The projects are built using SQL to solve real-world business problems and generate actionable insights from data.

## Skills Demonstrated
- Database Design
- Data Cleaning
- Data Analysis
- Joins
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- Aggregate Functions
- Views
- Stored Procedures
- Query Optimization
